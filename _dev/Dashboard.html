  └── Positioner

                    └── Head, Hair, Chest, HandRight ...

⚠️ Player body scale이 0.1이므로 CharacterMaker localScale을 10으로 오버라이드해야 실제 크기 1.0

상태명Player.cs 호출비고

파라미터: AttackSpeed float (default 1) — WhenAttackSpeedChanged()에서 SetFloat() 실시간 업데이트

  • EquipParts(PartsType, int index) — 파츠 교체
  • UnequipParts(PartsType) — 파츠 제거 (canToggle인 경우만)
  • SetColor(ColorTargetType, Color) — 색상 변경 (Skin / Hair / Beard)
  • ToPresetItem() — 현재 외형 직렬화 → LocalSaveManager 저장용
  • ApplyPresetItem(item) — 저장된 외형 복원
  • HandRight 무기 그룹 (동시에 하나만 활성): Sword, Axe, Bow, Wand, Staff, Spear, Blunt, Crossbow
  • 자동 연동: Bow/Crossbow 장착 시 Arrow/Bolt 표시 · Helmet 장착 시 HelmetHair 자동 교체
